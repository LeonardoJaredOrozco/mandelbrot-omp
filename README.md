## Descripcion
Generacion de imagen 8k del conjunto de Mandelbrot con filtro Gaussiano, 
optimizado progresivamente con OpenMP

## Compilacion (secuencial)
```bash
g++ -02 -o mandelbrot_seq src/mandelbrot_seq.cpp
./mandelbrot_seq
```
